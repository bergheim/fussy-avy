* fussy-avy

Typo-tolerant avy jumping. Like =avy-goto-char-timer=, but it doesn't abort when you mistype.

Type "footar" and it still finds "foobar". Space matches any non-word char, so "with " matches "with-".

** Installation

#+begin_src elisp
(use-package fussy-avy
  :ensure (:host github :repo "bergheim/fussy-avy")
  :bind (("C-'" . fussy-avy-goto-char-timer)
         :map evil-normal-state-map
         ("g " . evil-fussy-avy-goto-char-timer))
  :config
  ;; (setq fussy-avy-max-distance 2)      ; max edit distance (default 2)
  ;; (setq fussy-avy-min-input-length 3)  ; chars before fuzzy kicks in (default 3)
  ;; (setq fussy-avy-all-windows t)       ; search all windows (default t)
  ;; (fussy-avy-max-forgiving)            ; crank it up: distance=4, min-length=2
  )
#+end_src

** Faces

- =fussy-avy-match-exact= - exact matches
- =fussy-avy-match-fuzzy= - fuzzy matches (orange by default)

** Consult

=consult-fussy-avy= provides a completing-read interface to visible symbols.

** Dependencies

- Emacs 27.1+
- [[https://github.com/abo-abo/avy][avy]]
- [[https://github.com/minad/consult][consult]] (optional)
